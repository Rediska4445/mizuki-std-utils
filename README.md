# Ebanina Std. - Utils

---

## Описание

Это внутренний функциональный модуль проекта Ebanina,
который используется для общих задач в Mizuka

---

# Архитектура проекта

- network
- loggining
- collections
- weakly
- concurrency

---

## Начало работы

Чтобы использовать модуль, достаточно подключить его в _class-path_.
Модуль уже содержит в себе зависимости.

---

## Функциональность

- Работа с коллекциями
- Сетевое взаимодействие
- Параллелизм
- Логирование

---

*Использование модуля как что-то действительно рабочее, не подразумевается.*
